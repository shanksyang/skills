# TTS 引擎详细对比

## 引擎总览

| 特性 | Edge-TTS | 腾讯云 TTS | gTTS |
|------|----------|-----------|------|
| 费用 | 免费 | 按量计费 | 免费 |
| API Key | 不需要 | 需要 | 不需要 |
| 粤语专用音色 | 3 个 | 1 个 | 1 个 |
| 音质 | ★★★★ | ★★★★★ | ★★★ |
| 语速调节 | 支持 | 支持 | 仅慢速 |
| SSML 支持 | 支持 | 支持 | 不支持 |
| 网络要求 | WebSocket | HTTPS | HTTPS (Google) |
| 单次字数限制 | 无明确限制 | 150 字 | 5000 字节 |

## Edge-TTS 详情

### 技术原理
使用 WebSocket 协议连接 Microsoft Edge 浏览器的在线 TTS 服务，无需 Edge 浏览器、Windows 系统或 API Key。

### 粤语音色

| 音色 ID | 性别 | 风格 | 推荐场景 |
|---------|------|------|----------|
| `zh-HK-HiuGaaiNeural` | 女 | 活泼清晰 | 通用对话、教学 |
| `zh-HK-HiuMaanNeural` | 女 | 温柔沉稳 | 旁白、阅读 |
| `zh-HK-WanLungNeural` | 男 | 稳重大气 | 新闻播报、解说 |

### 调节参数
```yaml
tts:
  type: edge
  voice: zh-HK-HiuGaaiNeural
  rate: "+10%"     # 语速: -50% ~ +100%
  volume: "+0%"    # 音量: -50% ~ +100%
  pitch: "+0Hz"    # 音调: -50Hz ~ +50Hz
```

## 腾讯云 TTS 详情

### 粤语音色

| VoiceType | 名称 | 性别 | 类型 |
|-----------|------|------|------|
| 101019 | 智彤 | 女 | 精品音色 |

### API 参数
- `PrimaryLanguage`: 1（中文）
- `SampleRate`: 8000 / 16000（推荐）
- `Codec`: mp3 / wav / pcm
- `ModelType`: 1
- 单次请求限制: 150 字（超出需分段）

### 费用
- 免费额度：每月 80 万字符
- 超出后按量计费

## gTTS 详情

### 语言代码
- `zh-TW`: 台湾中文（接近粤语声调）
- `yue`: 粤语（部分版本支持）

### 限制
- 需要访问 Google Translate 服务
- 无法精细调节语速、音调
- 仅支持 MP3 输出
- 音质相对其他引擎较低

## 推荐选择

| 场景 | 推荐引擎 | 原因 |
|------|----------|------|
| 快速体验/免费使用 | Edge-TTS | 零成本，音质好 |
| 商业/高质量需求 | 腾讯云 TTS | 专业粤语音色 |
| 简单集成/教学 | gTTS | 最简单的 API |
| 无法连外网 | 腾讯云 TTS | 国内网络友好 |
